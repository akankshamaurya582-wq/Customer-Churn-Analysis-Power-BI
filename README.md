# 📊 Customer Churn Analysis Dashboard

## Overview
This project features an **interactive Power BI dashboard** designed to analyze **customer churn**. It highlights key metrics and visualizations to help understand customer behavior and retention patterns.

## 🔑 Key KPIs
- Total Users  
- Churn Rate  
- Customers by Product, Gender, Country, Activity  
- Customers with Credit Card  

## 📈 Visualizations
- Bar graph of **Churned Customers by Age Group**  
- Bar graph of **Customers by Balance**  
- Bar graph of **Customer Credit Score**  

## 🎯 Purpose
- Identify churn drivers  
- Provide actionable insights for retention strategies  
- Enable interactive exploration of customer data  

## 🛠️ Tools
- **Power BI** for dashboard creation and visualization  

---

Clone the repo, open the `.pbix` file in **Power BI Desktop**, and explore the dashboard interactively.
